
# React

## Detailed Breakdown

### 1. React Basics
- What is React? Why use it?
- Creating a new React app (using Create React App or Vite)
- File structure overview

#### Step-by-Step Exercise: First Component
1. Create a new React app.
2. Edit `App.tsx` or `App.js` to display "Hello, React!".
3. Create a new component called `Greeting` that takes a `name` prop and displays a personalized message.

### 2. State and Props
- useState hook
- Passing data via props

#### Step-by-Step Exercise: Counter
1. Add a button to your app that increases a counter when clicked.
2. Display the current count in the UI.
3. Pass the count as a prop to a child component.

### 3. Event Handling
- Handling click and input events

#### Step-by-Step Exercise: Controlled Input
1. Add an input field to your app.
2. Store its value in state.
3. Display the value below the input as the user types.

### 4. useEffect Hook
- Side effects and data fetching

#### Step-by-Step Exercise: Fetch Data
1. Use `useEffect` to fetch data from an API when the component mounts.
2. Display the data in your component.

### 5. Mini-Project: Weather Dashboard
Build a weather dashboard that fetches and displays weather data for a user-specified city. Use state, props, and effects.
