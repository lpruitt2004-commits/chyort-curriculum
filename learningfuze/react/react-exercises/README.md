# React Exercises: Granular Practice

1. Create a functional component that displays a greeting.
2. Use useState to toggle text between "ON" and "OFF".
3. Build a counter component with increment/decrement buttons.
4. Pass props from a parent to a child component.
5. Render a list of items from an array using map.
6. Use useEffect to fetch and display data from an API.
7. Create a form with controlled inputs and submit handling.
8. Conditionally render content based on state.
9. Use useRef to focus an input field on mount.
10. Build a simple modal component that opens/closes.
11. Create a custom hook for toggling boolean state.
12. Use Context to share state between components.
13. Implement a dark mode toggle in a React app.
14. Optimize a component with React.memo.
15. Add basic routing with React Router.

# Challenge
- Build a small dashboard app using several of the above concepts.
