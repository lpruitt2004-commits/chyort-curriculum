
# Node.js & Express

## Detailed Breakdown

### 1. Node.js Basics
- What is Node.js?
- Running JavaScript outside the browser

#### Step-by-Step Exercise: Hello, Node!
1. Create a file called `hello.js`.
2. Write a script that prints "Hello from Node.js!" to the console.
3. Run it with `node hello.js`.

### 2. Setting Up Express
- Installing Express
- Creating a basic server

#### Step-by-Step Exercise: Basic Express Server
1. Run `npm init -y` and `npm install express` in a new folder.
2. Create `index.js` and set up a server that responds with "Hello, Express!" on the root route.
3. Start the server and test it in your browser.

### 3. RESTful APIs
- What is a REST API?
- Creating GET and POST routes

#### Step-by-Step Exercise: Simple API
1. Add a `/greet` route that returns a JSON greeting.
2. Add a POST route that accepts a name and returns a personalized greeting.

### 4. Middleware
- What is middleware?
- Using built-in and custom middleware

#### Step-by-Step Exercise: Logger Middleware
1. Write middleware that logs each request to the console.
2. Use it in your Express app.
