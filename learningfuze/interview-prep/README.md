# Interview Prep

## Overview
- Types of interviews: technical, behavioral, system design
- Common coding challenge topics
- Whiteboarding tips

## Step-by-Step Exercises
1. Practice common coding challenges (FizzBuzz, anagrams, two-sum, etc.).
2. Mock behavioral interview questions and answers.
3. System design: design a URL shortener or chat app.
4. Whiteboard a solution to a given problem.
