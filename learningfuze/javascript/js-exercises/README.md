# JavaScript Exercises: Granular Practice

1. Declare variables of different types and log them to the console.
2. Write a function that returns the sum of two numbers.
3. Create an array of numbers and print only the even ones.
4. Write a function to reverse a string.
5. Use a for loop to print numbers 1-20.
6. Create an object representing a person and print their properties.
7. Add a click event to a button that changes the page background color.
8. Write a function that checks if a word is a palindrome.
9. Use setTimeout to display a message after 2 seconds.
10. Fetch data from a public API and log the result.
11. Add/remove items from an array and print the updated array.
12. Use map/filter/reduce on an array of numbers.
13. Write a function that returns the factorial of a number.
14. Create a simple calculator with add, subtract, multiply, divide functions.
15. Build a to-do list app with add, complete, and remove features.

# Challenge
- Combine several of the above into a small interactive app.
