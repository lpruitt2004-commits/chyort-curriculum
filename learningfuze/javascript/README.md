
# JavaScript

## Detailed Breakdown

### 1. JavaScript Basics
- Variables (var, let, const)
- Data types (string, number, boolean, array, object)
- Operators (arithmetic, comparison, logical)

#### Step-by-Step Exercise: Hello, JavaScript!
1. Create a new file called `hello.js`.
2. Declare a variable for your name and assign it a value.
3. Print a greeting to the console using your variable.
4. Try changing the variable and see the result.

### 2. Functions and Scope
- Function declarations and expressions
- Parameters and return values
- Scope (global, local)

#### Step-by-Step Exercise: Simple Functions
1. Write a function that takes a number and returns its square.
2. Call the function with different numbers and print the results.
3. Write a function that greets a user by name.

### 3. Arrays and Objects
- Creating arrays and objects
- Accessing and modifying elements/properties
- Array methods (push, pop, map, filter)

#### Step-by-Step Exercise: Array Practice
1. Create an array of your favorite foods.
2. Add a new food to the array.
3. Remove the last food from the array.
4. Print the array length.

### 4. DOM Manipulation
- Selecting elements
- Changing content and styles
- Handling events

#### Step-by-Step Exercise: Interactive Button
1. Create an HTML file with a button and a paragraph.
2. Write a script that changes the paragraph text when the button is clicked.

### 5. ES6+ Features
- Arrow functions
- Template literals
- Destructuring

#### Step-by-Step Exercise: Modern JavaScript
1. Rewrite a function using arrow syntax.
2. Use template literals to build a string.
3. Destructure an object or array.

### 6. Mini-Project: Interactive To-Do List
Combine your skills to build a to-do list app that lets users add, complete, and remove tasks. Store the list in local storage.
