
# SQL

## Detailed Breakdown

### 1. SQL Basics
- What is SQL?
- Tables, rows, and columns

#### Step-by-Step Exercise: First Table
1. Create a new database (use SQLite, MySQL, or PostgreSQL).
2. Create a table called `users` with columns for id, name, and email.
3. Insert a few sample users.
4. Write a query to select all users.

### 2. Basic Queries
- SELECT, INSERT, UPDATE, DELETE

#### Step-by-Step Exercise: CRUD Operations
1. Write a query to update a user's email.
2. Write a query to delete a user by id.

### 3. Joins and Relationships
- One-to-many and many-to-many relationships
- JOIN queries

#### Step-by-Step Exercise: Join Practice
1. Create a `posts` table with a user_id column.
2. Write a query to select all posts with the user's name (using JOIN).

### 4. Database Design
- Normalization
- Primary and foreign keys

### 5. Mini-Project: Sample Database
Design and query a small database for a blog or to-do app. Practice writing queries for real-world scenarios.
