
# LearningFuze Curriculum Recreation

This directory contains modules, exercises, and projects to help you brush up and become a competent programmer again. Track your progress and add notes as you go!

## Curriculum Outline

### 1. HTML & CSS
- Semantic HTML5
- CSS fundamentals, Flexbox, Grid
- Responsive design
- Accessibility
- Mini-projects: Portfolio landing page, responsive layouts

### 2. JavaScript
- Variables, data types, operators
- Functions, scope, closures
- Arrays, objects, loops
- DOM manipulation
- ES6+ features (let/const, arrow functions, destructuring, etc.)
- Fetch API & AJAX
- Mini-projects: To-do list, calculator, interactive games

### 3. React
- Components, props, state
- Functional vs class components
- Event handling
- Hooks (useState, useEffect, custom hooks)
- Component lifecycle
- Routing (React Router)
- Mini-projects: Weather dashboard, blog, SPA

### 4. Node.js & Express
- Setting up a server
- RESTful API design
- Middleware
- Routing
- Connecting to databases (MongoDB, PostgreSQL, MySQL)
- Authentication basics
- Mini-projects: Simple REST API, CRUD app

### 5. SQL & Databases
- Relational database concepts
- Basic queries (SELECT, INSERT, UPDATE, DELETE)
- Joins, relationships, normalization
- Database design
- Mini-projects: Design/query a sample database

### 6. Algorithms & Data Structures
- Big O notation
- Sorting and searching algorithms
- Recursion
- Arrays, stacks, queues, linked lists, trees, hash tables
- Coding challenges and whiteboarding

### 7. Capstone & Group Projects
- Fullstack application (frontend + backend)
- Team collaboration (Git, GitHub)
- Agile workflow (sprints, standups)
- Deployment (Netlify, Heroku, Vercel, etc.)
- Demo day presentations

### 8. Additional Topics (optional/advanced)
- TypeScript basics
- Testing (Jest, Mocha, etc.)
- DevOps basics (CI/CD, Docker)
- Mobile-first & PWA concepts
- Interview prep & career support
