
# Projects

## Capstone & Group Projects
- Plan, design, and build a fullstack application (frontend + backend)
- Work in teams using Git and GitHub
- Use Agile workflow (sprints, standups)
- Prepare for demo day presentations

## Tips for Project Success
- Start with wireframes and user stories
- Break work into small, manageable tasks
- Commit early and often
- Write clear documentation (README, comments)
- Test your code and get feedback

## Project Ideas
- Portfolio website
- Blog platform
- To-do or productivity app
- Weather dashboard
- Chat or messaging app

Document your process and outcomes for each project. Include screenshots, setup instructions, and lessons learned.
