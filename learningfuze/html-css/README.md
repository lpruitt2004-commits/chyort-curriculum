
# HTML & CSS

## Detailed Breakdown


### 1. HTML Basics
1. Structure of an HTML document (doctype, html, head, body)
2. Headings, paragraphs, links, images, lists
3. Forms and input elements

#### Assignment 1: Basic Web Page
1. Create a new file called `index.html`.
2. Add the basic HTML boilerplate (doctype, html, head, body).
3. Add your name as a heading.
4. Add a short bio as a paragraph.
5. Add a link to your favorite website.
6. Add an image (use a placeholder or your own).
7. Create an unordered list of your hobbies.


### 2. CSS Fundamentals
1. Selectors, properties, and values
2. Colors, fonts, margins, padding, borders
3. Classes and IDs

#### Assignment 2: Styling Your Page
1. Create a new file called `styles.css`.
2. Link it to your `index.html` in the `<head>`.
3. Change the background color of the page.
4. Set a custom font for your text.
5. Add padding and margin to your elements.
6. Style your heading and list with different colors.


### 3. Layouts: Flexbox & Grid
1. Flexbox basics (container, direction, justify, align)
2. Grid basics (rows, columns, gap)

#### Assignment 3: Flexbox Navigation Bar
1. In your `index.html`, add a `<nav>` with links for Home, About, Projects, Contact.
2. Use Flexbox in `styles.css` to arrange the links horizontally.
3. Make the navigation bar responsive (stack vertically on small screens).


### 4. Responsive Design
1. Media queries
2. Mobile-first approach

#### Assignment 4: Responsive Card
1. Add a `<div>` with class `card` containing a title, image, and description.
2. Use CSS to style the card with padding, border, and shadow.
3. Add a media query to make the card full-width on screens smaller than 600px.


### 5. Mini-Project: Personal Portfolio Landing Page
Assignment 5: Combine all the above skills to build a simple, responsive portfolio page with your name, bio, links, and a project showcase section.
