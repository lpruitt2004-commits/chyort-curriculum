# CSS Exercises: Granular Practice

1. Style a heading and paragraph with different colors and fonts.
2. Add a background image to a section and adjust its size/position.
3. Create a navigation bar using Flexbox.
4. Build a responsive grid layout for a photo gallery.
5. Use media queries to make a layout mobile-friendly.
6. Add hover effects to buttons and links.
7. Style a form with custom input, label, and button styles.
8. Create a card component with box-shadow and border-radius.
9. Use CSS variables for a color theme.
10. Animate an element with keyframes (e.g., fade in/out).
11. Create a sticky header that stays at the top on scroll.
12. Use the :nth-child selector to style list items differently.
13. Build a progress bar with CSS only.
14. Style a table with alternating row colors and hover effects.
15. Implement a dark mode toggle using CSS classes.

# Challenge
- Combine several of the above into a single, styled landing page.
