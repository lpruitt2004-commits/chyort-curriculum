# HTML Exercises: Granular Practice

1. Create a basic HTML page with a heading and paragraph.
2. Add an image with alt text and a caption.
3. Create an ordered and unordered list of your favorite foods and hobbies.
4. Build a table showing your weekly schedule.
5. Add a form with text, email, and password inputs, plus a submit button.
6. Use semantic tags (header, nav, main, section, article, aside, footer) to structure a blog post.
7. Link to an external website and open it in a new tab.
8. Embed a YouTube video using the iframe tag.
9. Add a blockquote and cite the source.
10. Create a contact form with textarea and radio buttons.
11. Add a favicon to your HTML page.
12. Use meta tags for description and keywords.
13. Create a navigation bar with anchor links to sections on the same page.
14. Add a download link for a PDF file.
15. Use the details and summary tags for a collapsible FAQ section.

# Challenge
- Combine all the above into a single, well-structured HTML page.
