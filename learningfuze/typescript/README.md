## Step-by-Step Exercises
1. Install TypeScript globally and locally in a project.
2. Create a `tsconfig.json` and understand its options.
3. Write a simple `hello.ts` and compile it.
4. Practice with types: string, number, boolean, array, tuple, enum, any, void, never.
5. Functions with typed parameters and return values.
6. Interfaces and type aliases.
7. Classes and inheritance in TypeScript.
8. Type assertions and type guards.
9. Migrating a small JS file to TypeScript.

- Convert a small JavaScript app (e.g., to-do list) to TypeScript.
